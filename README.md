# SPChat
A simple chatting web app that utilizes [Socket.IO](https://socket.io/).

## Installation

 1. Clone this repo
 2. `cd` into the directory
 3. run `$ npm install`
 4. run `$ npm start`
 5. Point your browser to whatever the console shows.
